## Model

The model is defined by four axioms (P1–P4):

- **P1** — Causal order: L[e] = max{L[parent]} + 1
- **P2** — Sequential growth: one new event per step, P causal parents
- **P3** — Stochastic locality: Boltzmann-weighted parent selection, σ = α · Lmax
- **P4** — Self-consistency: α determined by Alocal = 0

Key parameters:
- α ≈ 0.037 (numerically confirmed)
- P = 4 (geometrically constrained by Simplex Condition)
- N* ≈ 100,000 (characteristic scale)
